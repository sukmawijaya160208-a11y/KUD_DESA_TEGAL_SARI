# Log Skill — KUD Desa Sari Subur

Gunakan skill ini untuk mencatat atau membaca progres development.

## Cara pakai
- "catat progres" — tulis sesi saat ini
- "cek log" — baca catatan sebelumnya
- "update log" — tambah catatan baru

## Format catatan
```markdown
## [YYYY-MM-DD] — Sesi #
- **Yang dikerjakan**: ...
- **Keputusan**: ...
- **Kendala**: ...
- **Next**: ...
```
